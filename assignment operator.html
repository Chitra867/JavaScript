assignment operator is used to assigned a value to the variable.
        = means value assign
        += eg: x+=y is same as x=x+y
        -= eg: x-=y is same as x=x-y
        *= eg: x*=y is same as x=x*y
        /= eg: x/=y is same as x=x/y
        **= eg: x**=y is same as x=x**y
        %= eg: x%=y is same as x=x%y
